## 禁止事項

正式站 smoke **禁止**以下操作：

| 禁止 | 原因 |
|------|------|
| ❌ **Permanent delete** 真實客戶 | 不可恢復 |
| ❌ **執行 purge**（cron 手動觸發或批量清理） | 硬刪除生產資料 |
| ❌ **隨意刪除正式員工** | 客戶會轉移；歷史審計影響 |
| ❌ **執行 `npm run db:migrate:remote`** | 需維護窗口與備份 |
| ❌ **直接改 D1 / R2** | 繞過應用邏輯 |
| ❌ **在 prod 測 merge_customers 審批通過** | 執行邏輯為 placeholder |
| ❌ **使用 `@crm.local` 測試帳號** | 僅限本地 seed |

### 測試資料命名規範

新增測試客戶請使用可識別前綴，方便事後清理：

```
[SMOKE] 測試客戶 - {日期} - {測試人 initials}
```

必填欄位請填完整（含 **項目名稱 requestedProjectName**），避免 validation 400。
